üîë **Key Features**

* **NLP Sentiment Polarity:** Utilizes TextBlob to calculate accurate sentiment scores from student feedback, ensuring a precise understanding of campus morale beyond simple star ratings.
* **Thematic Keyword Analysis:** Integrated text processing to isolate top-frequency keywords (e.g., "**Teachers**," "**Skills**," "**Opportunities**"), identifying the primary drivers of student satisfaction.
* **Detailed Performance Heatmap:** Developed a transposed, high-resolution heatmap that maps 20+ survey questions against multiple courses to isolate department-specific "danger zones".
* **Actionable Gap Identification:** Delivers granular, data-backed insights that clearly identified critical priorities like **ICT tool usage** and **Mentoring for emotional growth**, both scoring **3.58/5.0**.
* **Data Model & Preparation:** Developed a resilient script that parses complex survey strings (e.g., "3.00/60.00") into clean float metrics for rigorous statistical analysis.
